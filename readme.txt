== Description ==


WooCommerce Product Variation Table & Variation Quick Cart comes to make add to cart process faster for variable products. Most of the cases before adding a variation to cart there are some many clicks needed, which is time consuming. As a result most customers feel disappointed to choose the right variation. Ultimately business losses.

WooXperto is introducing a Product Variation table and quick cart plugin to solve this issue.


== Installation ==
1. Upload the Plugin Files: Upload the Product Variation With Quick Cart folder to the `/wp-content/plugins/` directory.
   2.Activate the Plugin: Activate the plugin through the \'Plugins\' menu in WordPress.
3. Install Required Plugins: Make sure WooCommerce is installed and activated.
4. Configure the Plugin: Go to \'Variation Table\' Settings\' under Admin Menu to configure the plugin

== Frequently Asked Questions ==
1. How Product Variation Table with Quick Cart work?
